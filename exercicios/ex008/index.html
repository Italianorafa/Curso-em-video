Principais formatações

Negrito / Destaque

Nesta frase temos um termo em negito usando a tag B.

Nesta outra, temos um termo em negrito usando STRONG.

Itálico / Ênfase

Nesta frase temos um termo em itálico usando a tag I (não semântica).

Nesta frase temos um termo em ênfase usando a tag EM(semântica)

Texto marcado

Podemos criar também um texto marcado usando a tag MARK

Texto grande e pequeno

Estamos criando um texto grande, com a ta BIG (obsoleta), e um texto pequeno, com a tag SMALL.

Texto excluido/deletado

Podemos indicar um texto como excluído para indicar que ele deve ser desconsiderado pelo leitor, usando a tag DEL

Texto inserido

Podemos marcar um texto como inserido para dar ênfase e indicar que foi adicionado, usando a tag INS

Pode-se usar a tag U para esse mesmo fim, porém é sem semântica.

Texto sebrescrito

Para textos do tipo x20 + 3, usa-se SUP

Texto subscrito

Para textos do tipo H20, usa-se SUB
