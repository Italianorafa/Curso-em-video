Tags em HTML5

Em HTML5 podemos inserir os elementos da lista a seguir usando tags. Passe o mouse por cima do elemento para revelar a tag.

  • Parágrafos
  • Títulos
  • Imagens
  • Abreviações
  • Códigos-fonte
  • Listas numeradas
  • Listas com demarcadores
  • Textos subscritos
  • Textos sobrescritos
